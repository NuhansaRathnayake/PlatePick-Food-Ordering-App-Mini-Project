# PlatePick - Food Ordering App 

## Project Description
PlatePick is a simple and user-friendly Android application designed for ordering food. The app allows users to easily browse a categorized menu, add their favorite meals to a cart, and place an order. This project specifically focuses on the core food ordering process and does not include a real-time payment gateway (supports Cash on Delivery and manual Card selection).

## Features
* **User Authentication:** Secure Login and Registration with email and strong password validation.
* **View Menu:** Browse different food categories like Burgers, Pizzas, and Drinks.
* **Add to Cart:** Select items from the menu and view them in a dedicated cart page.
* **Place Order:** Checkout process with delivery address and phone number validation. 
* **Order History:** View previously placed orders saved securely in the local database.

## Technologies Used
* **Programming Language:** Java
* **UI / Layouts:** XML (Material Design Components)
* **Database:** SQLite (To store user details and order history)
* **Development Environment:** Android Studio
